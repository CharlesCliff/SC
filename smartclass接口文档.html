smartclass接口文档

接口名称

  • url
  • 参数
  • 返回
  • 示例

1.1借口名称:获取所有项目

  • url student/course/project/$
  • 参数 coursename 课程名称
  • 返回 {'result':1,'message':'get project list success','projectlist':projectlist,'studentlist':studentlist}
  • 示例 smartclass/student/course/project/$?coursename=网络实习

1.2接口名称》提交测试答案

  • url student/course/test/answer/$
  • 参数 coursename testname answers[]
  • 返回 result:0失败 1成功{'result':1,'message':'save answer success'}
  • 示例 smartclass/student/course/test/answer/?testname=testname&coursename=coursename&answers=A,B,C,A,D

1.3接口名称 :提交分组信息

  • url student/course/project/groupinfo/$
  • 参数 coursenam,projectname,memberlist(array)组队成员列表,questionlist(array)所选题目列表（有序）
  • 返回 {'result':1,'message':'add groupinfo success'}result:0failed 1succeded
  • 示例 smartclass/student/course/project/groupinfo/?coursename=coursename&projectname=projectname&memberlist=2,5,15&questionlist=3,2,4

1.4接口名称:获取课程的所有题目

  • url student/course/project/question/
  • 参数 coursename,projectname
  • 返回{'result':1,'message':'get project question success','questionlist':pq}
  • 示例 smartclass/student/course/project/question

1.5接口名称:学生注册

  • url student/register/$
  • 参数 username,password
  • 返回 {'result':1,'message':'regster success'}
  • 示例 /smartclass/student/register/?username=chenyang&password=12345
  • ###1.6接口名称 :学生登陆
  • url student/login/$
  • 参数 username,password
  • 返回 {'result':1,'message':'login success'}
  • 示例/student/login?username=username&password=password

1.7接口名称:获取数据库所有课程

  • url student/allcourse/
  • 参数 null
  • 返回 allcourselist:[] 所有课程的名称集合
  • 示例 smartclass/student/allcourse/ *

1.8接口名称:学生自己所添加课程列表

  • url student/mycourse/$
  • 参数 none
  • 返回 mycourselist:[]
  • 示例 student/mycourse/

1.9接口名称:学生加入课程

  • url student/joincourse/$
  • 参数 coursename
  • 返回 {'result':1,'message':'join course success'}
  • 示例 student/joincourse/?coursename=coursename
